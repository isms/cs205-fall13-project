CS 205 Project

Parallelizing an ILP Assignment Problem

About

We model an assignment problem in a small private school wherein each grade of 80 to 100 students must be divided into four classes. In assigning students to classes, the school aims to satisfy several constraints such as not placing certain pairs of students in the same class or keeping boy/girl ratios balanced. In previous work, a model was developed to assist with creating class placements. Following the constraints and objectives given by administrators at the school, the problem is implemented as:

  1. An Integer Linear Program (ILP) for satisfying placement constraints.
  2. Non-linear heuristics to further improve on the outputs of the ILP model.
The overarching limitation of the model is that creating feasible solutions is a lengthy process. We wish to improve on this by parallelization.

Problem statement

Can we take this serial implementation and parallelize the problem so that it can be split among many processors?

Results

WWe achieved massive speedups (over 100x) by parallelizing this problem using MPI, going from the order of almost 4 hours down to about 2 minutes using 256 processors and work splitting. MPI proved a powerful and flexible way to parallelize LP solving without significantly modifying the serial model. You can find out more about the our results here.

D. Newman, I. Slavitt, 2013
